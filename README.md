# Expense Tracker API

A REST API for tracking personal expenses, built with FastAPI and SQLite. This is a web API rebuild of an earlier CLI project, built to learn backend development, databases, and authentication.

## Features

- User registration and login with JWT-based authentication
- Passwords are securely hashed with bcrypt, never stored in plain text
- Full CRUD for expenses: create, read, update, and delete
- Each user can only access their own expenses
- Interactive API documentation auto-generated by FastAPI (Swagger UI)

## Tech stack

- **FastAPI** — web framework
- **SQLAlchemy** — ORM for database access
- **SQLite** — database
- **Passlib (bcrypt)** — password hashing
- **python-jose** — JWT token creation and verification

## How to run

1. Clone this repository
2. Create and activate a virtual environment:
   \`\`\`
   python -m venv venv
   source venv/Scripts/activate
   \`\`\`
3. Install dependencies:
   \`\`\`
   pip install -r requirements.txt
   \`\`\`
4. Run the server:
   \`\`\`
   uvicorn main:app --reload
   \`\`\`
5. Open your browser to `http://127.0.0.1:8000/docs` to explore and test the API

## API overview

| Method | Endpoint       | Description                              |
| ------ | -------------- | ---------------------------------------- |
| POST   | /register      | Create a new user account                |
| POST   | /login         | Log in and receive an access token       |
| POST   | /expenses      | Add a new expense (requires auth)        |
| GET    | /expenses      | Get all of your expenses (requires auth) |
| PUT    | /expenses/{id} | Update an expense (requires auth)        |
| DELETE | /expenses/{id} | Delete an expense (requires auth)        |
